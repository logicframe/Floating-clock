# Floating-clock
Just a small floating clock, shows current time.

By default it's dark purple.(`#6a329f`) As my experience has shown, this is an excellent color, clock was always visible.

# **So, how to use:**
1. Launch the clock
<img width="663" height="72" alt="изображение" src="https://github.com/user-attachments/assets/7d002c89-a0f5-4f4e-8492-0d4512bcc8af" />

2. Drag it(you have to hover your mouse directly over clock, not on the gaps between numbers) to whatever place you want to


To close the clock, just close the consome(cmd).
<img width="993" height="557" alt="изображение" src="https://github.com/user-attachments/assets/53a91fce-3564-4ad0-9e06-8684bf683e56" />


Uses about 20 megabytes of RAM. May need less if you have Python installed(as for me it was 16 megabytes)
